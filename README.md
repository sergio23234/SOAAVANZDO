# AyD1ProyectoClaseApi
BACKEND PROYECTO CLASE AYD1

### Compilar

Se puede compilar usando el comando:

```sh
$ mvn package
```

### Ejecutar

Se puede ejecutar usando el comando:

```sh
$ mvn spring-boot:run
```

La aplicacion puede ser detenida presionando <kbd>ctrl</kbd>+<kbd>c</kbd> en la consola.
